# Synthia - помощник для музыкальных плейлистов

## Запуск

`npm run dev` - запуск в режиме разработки

## Стек

- `Grammy` - фреймворк для бота
- `Fastify?` - фреймворк для бэка
- `Postgres/SQLite?` - база данных
- хранилище файлов
- хостинг

## План

Команды:

- Добавить песню
- Создать плейлист
- Настройки

### Добавление песни

- Загрузка файла
- Проверка на повторное добавление
- Обработка, сохранение имени в памяти
- Сохранение файла в облаке
